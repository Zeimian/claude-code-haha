# baoyu-sklls 快速入门指南

## 系统简介

baoyu-sklls（宝玉技能系统）是一个基于毛泽东思想指导原则的全能内容创作系统。它将现代AI技术与实践哲学相结合，提供了从文案写作到视觉设计、从智能配图到知识可视化的全方位内容创作能力。

## 核心理念

系统遵循毛泽东思想指导原则：
- **实事求是**: 根据实际需求创作内容
- **矛盾论**: 识别并解决创作中的主要矛盾
- **实践论**: 通过实际应用验证效果
- **群众路线**: 以目标受众为中心
- **游击战术**: 快速响应不同需求

## 快速开始

### 1. 内容创作（新手推荐）

从简单的文案创作开始：

```bash
/writing-copywriting-agent blog-post "写一篇关于人工智能发展前景的博客"
```

### 2. 智能配图

为你的内容自动添加插图：

```bash
/intelligent-article-illustrator my-blog-post.md --style tech
```

### 3. 并行创作

同时进行文案和视觉创作：

```bash
/parallel-copywriting-agents social-media "为新产品创建社交媒体内容"
```

## 使用技巧

### 1. 组合使用
将多个技能组合使用以获得最佳效果：

```bash
# 步骤1: 创建内容
/writing-copywriting-agent article "写一篇关于气候变化的文章"

# 步骤2: 添加插图
/intelligent-article-illustrator climate-article.md --style nature

# 步骤3: 优化视觉
/visual-illustration-agent climate-article "为气候文章创建视觉规范"
```

### 2. 指定参数
使用具体参数获得更精确的结果：

```bash
/parallel-copywriting-agents [内容类型] "[具体要求]"
```

例如：
```bash
/parallel-copywriting-agents marketing "为环保产品创建营销内容，风格温馨亲和"
```

### 3. 小红书专项
针对小红书平台优化：

```bash
/xiaohongshu-content-agent beauty-tips "创建美容护肤技巧内容，6张图，温馨可爱风格"
```
