# 毛泽东思想指导下的并行文案与插图生成系统

## 新增功能

本项目新增了基于毛泽东思想指导原则的并行内容创作系统，包含以下几个专门的 Agent：

### 1. 并行文案代理系统 (`/parallel-copywriting-agents`)
- 同时调动文案写作和插图生成两个专业代理
- 体现"集中优势兵力"的战略思想
- 实现内容创作效率最大化
- **新增**：集成智能文章配图规划功能

### 2. 专业文案写作代理 (`/writing-copywriting-agent`)
- 创造人性化、个性化风格的中文文案
- 去除AI味道，使内容更自然
- 适应不同文体和语境需求
- **新增**：识别内容中适合插图的位置

### 3. 专业视觉插图代理 (`/visual-illustration-agent`)
- 为文案内容创建配套视觉说明
- 确保视觉元素与文本内容和谐统一
- 考虑文化敏感性和受众偏好

### 4. 综合文案视觉代理 (`/comprehensive-copywriting-agent`)
- 同步创建文案和视觉内容
- 确保图文内容高度匹配
- 优化整体传播效果
- **新增**：集成插图推荐功能

### 5. 智能文章配图代理 (`/intelligent-article-illustrator`)
- **新增功能**：自动分析文章结构并识别配图位置
- 根据内容自动匹配9种预设插图风格
- 生成详细插图提示词并调用图像生成工具
- 将生成的插图插入到合适位置

### 6. 小红书内容创作代理 (`/xiaohongshu-content-agent`)
- **新增功能**：专业为小红书平台创作内容
- 拆解复杂内容为小红书风格的系列信息图
- 适配小红书用户的阅读习惯和审美偏好
- 生成适合小红书的文案和视觉内容

## 智能配图功能详解

### 9种预设风格
1. **tech**（科技感）- 适用于AI、技术、数据类内容
2. **warm**（温暖亲和）- 适用于情感、生活、成长类内容
3. **minimal**（极简）- 适用于商业、设计、效率类内容
4. **playful**（趣味涂鸦）- 适用于教育、娱乐、轻松话题
5. **notion**（线稿风）- 适用于笔记、教程、概念说明
6. **elegant**（优雅精致）- 适用于时尚、艺术、高端品牌
7. **vibrant**（活力四射）- 适用于运动、音乐、年轻化内容
8. **retro**（复古怀旧）- 适用于历史、文化、怀旧主题
9. **nature**（自然清新）- 适用于环保、健康、自然主题

### 自动匹配规则
- 包含 "AI", "算法", "数据", "技术" → tech 风格
- 包含 "个人成长", "情感", "生活", "心灵" → warm 风格
- 包含 "效率", "工具", "管理", "商业" → minimal 风格
- 包含 "学习", "儿童", "游戏", "轻松" → playful 风格

## 小红书内容创作功能

### 设计理念
专门为小红书平台优化，符合小红书用户的阅读习惯和审美偏好：

1. **系列信息图拆解** - 将复杂内容拆解为1-10张系列信息图
2. **视觉风格规范** - 卡通风格、手绘元素、莫兰迪色系
3. **排版原则** - 信息精简、多留白、重点突出
4. **互动元素** - 加入emoji、贴纸、对话气泡等元素

### 小红书风格特点
- 竖版布局（3:4 或 9:16）
- 卡通手绘风格
- 柔和莫兰迪色系
- 手写字体效果
- 萤光笔划线强调
- 装饰性图标和贴纸

## 毛泽东思想指导原则

- **实事求是**: 基于实际内容需求和用户反馈优化内容
- **矛盾论**: 识别内容创作中的主要矛盾并优先解决
- **实践论**: 通过A/B测试和实际应用验证内容效果
- **群众路线**: 以用户/受众为中心设计内容
- **游击战术**: 快速响应、灵活调整内容策略

## 设计理念

系统设计遵循毛泽东思想中的"集中优势兵力"战略，通过并行处理提高工作效率。每个代理专注于特定任务，多代理协同工作解决复杂内容创作需求，体现了分工合作和统筹协调的思想。

新增的智能配图功能体现了"实事求是"原则，仅在确实能增强内容理解的地方添加插图，避免为配图而配图的形式主义。

小红书内容创作功能体现了"群众路线"原则，以小红书用户为中心，创建符合其审美和阅读习惯的内容。
