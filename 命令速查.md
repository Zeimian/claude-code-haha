# Claude Code Haha 中文命令速查手册

## 🚀 启动命令

```bash
# 启动交互式 TUI 界面（默认模式）
./bin/claude-haha

# 无头模式：单次问答后退出（适合脚本/CI）
./bin/claude-haha -p "你的问题"

# 管道输入
echo "解释这段代码" | ./bin/claude-haha -p
cat file.py | ./bin/claude-haha -p "请审查这段代码"

# 降级模式：简化交互（TUI 异常时使用）
CLAUDE_CODE_FORCE_RECOVERY_CLI=1 ./bin/claude-haha

# 指定模型启动
./bin/claude-haha --model qwen3-max-2026-01-23

# 继续上次对话
./bin/claude-haha -c

# 恢复指定会话
./bin/claude-haha -r

# 调试模式（显示详细日志）
./bin/claude-haha -d

# 精简模式（跳过 hooks/插件/LSP）
./bin/claude-haha --bare
```

## 🎮 TUI 内交互命令

在 TUI 界面中输入以下命令：

| 命令 | 说明 |
|------|------|
| `/model` | 切换模型（显示所有可用模型列表） |
| `/init` | 创建 CLAUDE.md 项目配置文件 |
| `/commit` | 提交代码到 Git |
| `/review` | AI 代码审查 |
| `/help` | 查看所有可用命令 |
| `/btw <问题>` | 不中断当前任务，问一个小问题 |
| `/clear` | 清空当前对话 |
| `/compact` | 压缩对话历史（节省 token） |
| `/cost` | 查看本次会话的 API 费用 |
| `/status` | 查看系统状态 |
| `/resume` | 恢复之前的会话 |
| `Esc` | 中断当前 AI 生成 |

## ⚙️ 启动参数详解

### 基础参数

| 参数 | 缩写 | 说明 |
|------|------|------|
| `--print` | `-p` | 无头模式，单次问答后退出 |
| `--debug` | `-d` | 开启调试模式 |
| `--verbose` | | 开启详细输出 |
| `--help` | `-h` | 显示帮助信息 |
| `--continue` | `-c` | 继续上次对话 |
| `--resume [ID]` | `-r` | 恢复指定会话 |
| `--name <名称>` | `-n` | 给会话设置名称 |

### 模型参数

| 参数 | 说明 |
|------|------|
| `--model <模型>` | 指定本次会话使用的模型 |
| `--thinking <模式>` | 思考模式：enabled / adaptive / disabled |
| `--fallback-model <模型>` | 主模型过载时的备用模型 |
| `--effort <级别>` | 努力程度：low / medium / high / max |

### 输入输出参数

| 参数 | 说明 |
|------|------|
| `--output-format <格式>` | 输出格式：text / json / stream-json |
| `--input-format <格式>` | 输入格式：text / stream-json |
| `--json-schema <规则>` | JSON Schema 结构化输出验证 |
| `--max-turns <次数>` | 最大对话轮数（仅 --print 模式） |

### 安全与权限

| 参数 | 说明 |
|------|------|
| `--dangerously-skip-permissions` | 跳过所有权限检查（⚠️ 仅限沙箱） |
| `--permission-mode <模式>` | 权限模式 |
| `--allowed-tools <工具>` | 允许使用的工具列表 |
| `--disallowed-tools <工具>` | 禁止使用的工具列表 |

### 高级参数

| 参数 | 说明 |
|------|------|
| `--bare` | 精简模式：跳过 hooks、LSP、插件等 |
| `--system-prompt <提示>` | 自定义系统提示词 |
| `--append-system-prompt <提示>` | 在默认提示词后追加内容 |
| `--mcp-config <配置>` | 加载 MCP 服务器配置 |
| `--settings <文件>` | 加载额外设置文件 |
| `--add-dir <目录>` | 添加额外允许访问的目录 |
| `--agent <代理>` | 指定使用的 Agent |
| `--agents <JSON>` | 定义自定义 Agents |
| `--plugin-dir <路径>` | 加载指定目录的插件 |
| `--session-id <UUID>` | 使用指定的会话 ID |
| `--ide` | 自动连接 IDE |
| `--chrome` | 启用 Chrome 集成 |

## 🔧 环境变量说明

| 变量 | 说明 |
|------|------|
| `ANTHROPIC_API_KEY` | API 密钥（x-api-key 头） |
| `ANTHROPIC_AUTH_TOKEN` | Auth Token（Bearer 头）⚠️ 与上面二选一 |
| `ANTHROPIC_BASE_URL` | API 端点地址 |
| `ANTHROPIC_MODEL` | 默认模型 |
| `ANTHROPIC_DEFAULT_SONNET_MODEL` | Sonnet 级别模型映射 |
| `ANTHROPIC_DEFAULT_OPUS_MODEL` | Opus 级别模型映射 |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL` | Haiku 级别模型映射 |
| `ANTHROPIC_SMALL_FAST_MODEL` | 小型快速模型（内部用） |
| `ANTHROPIC_CUSTOM_MODEL_OPTION` | 自定义模型（显示在 /model 菜单） |
| `API_TIMEOUT_MS` | API 超时时间（毫秒） |
| `DISABLE_TELEMETRY` | 禁用遥测（设为 1） |
| `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC` | 禁用非必要网络请求 |
| `CLAUDE_CODE_FORCE_RECOVERY_CLI` | 强制使用降级模式（设为 1） |

## 💡 使用技巧

1. **首次使用**：启动后输入 `/init` 创建项目配置
2. **切换模型**：输入 `/model` 打开模型选择器
3. **长对话卡顿**：输入 `/compact` 压缩历史
4. **查看费用**：输入 `/cost` 查看 token 消耗
5. **中断生成**：按 `Esc` 键立即停止
6. **继续对话**：下次启动时加 `-c` 参数
7. **批量处理**：用 `-p` 参数配合管道
