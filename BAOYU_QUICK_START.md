# baoyu-sklls 快速入门指南

欢迎使用毛泽东思想指导下的并行文案与插图生成系统（baoyu-sklls）！

## 系统简介

baoyu-sklls 是一个基于毛泽东思想指导原则的内容创作系统，它通过并行处理机制同时调动文案写作和插图生成两个专业代理，实现内容创作的高效协同。

## 核心技能

### 1. 并行文案代理系统 (`/parallel-copywriting-agents`)
同时处理文案和插图创作，体现"集中优势兵力"的战略思想。

**使用方法：**
```
/parallel-copywriting-agents [内容类型] [具体要求]
```

**示例：**
```
/parallel-copywriting-agents social-media "为新产品创建微信推广文案和视觉说明"
```

### 2. 专业文案写作 (`/writing-copywriting-agent`)
专注于创造人性化、个性化风格的中文文案，去除AI味道。

**使用方法：**
```
/writing-copywriting-agent [文案类型] [要求]
```

**示例：**
```
/writing-copywriting-agent product-description "写一份吸引年轻人的手机壳产品描述"
```

### 3. 专业视觉插图 (`/visual-illustration-agent`)
为文案内容创建配套的视觉说明，确保图文和谐统一。

**使用方法：**
```
/visual-illustration-agent [上下文] [视觉要求]
```

**示例：**
```
/visual-illustration-agent product-promotion "为手机壳产品图生成视觉说明 - 时尚、年轻化风格，明亮色彩"
```

### 4. 综合文案视觉 (`/comprehensive-copywriting-agent`)
同步创建文案和视觉内容，确保两者高度匹配。

**使用方法：**
```
/comprehensive-copywriting-agent [内容类型] [要求]
```

**示例：**
```
/comprehensive-copywriting-agent marketing-material "创建营销材料，包含文案和视觉规范"
```

## 毛泽东思想指导原则

1. **实事求是** - 基于实际需求和用户反馈创作内容
2. **矛盾论** - 识别内容创作中的主要矛盾并优先解决
3. **实践论** - 通过实际应用验证内容效果
4. **群众路线** - 以用户/受众为中心设计内容
5. **游击战术** - 快速响应、灵活调整内容策略

## 最佳实践

- **明确需求** - 在使用技能前明确内容类型、目标受众和期望风格
- **适当并行** - 仅在需要同步处理文案和视觉时使用并行系统
- **迭代优化** - 根据反馈持续改进内容质量和效果
- **质量检查** - 确保生成内容符合目标平台和受众要求
