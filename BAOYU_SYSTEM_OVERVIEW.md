# 毛泽东思想指导下的全能内容创作系统 (baoyu-sklls)

## 系统概述

本系统是一个综合性的AI内容创作平台，基于毛泽东思想指导原则构建，旨在实现高效、智能、可进化的内容创作流程。系统集成了文案写作、视觉设计、智能配图、平台发布和漫画脚本生成等功能，特别优化了小红书等内容平台的创作流程。

## 核心理念

### 毛泽东思想指导原则

1. **实事求是** (Seek truth from facts)
   - 根据实际内容需求和用户反馈优化创作
   - 基于真实数据调整创作策略
   - 避免形式主义的内容填充

2. **矛盾论** (Theory of contradictions)
   - 识别内容创作中的主要矛盾（如信息密度 vs 可读性）
   - 优先解决影响内容质量的核心问题
   - 平衡不同创作要素之间的关系

3. **实践论** (Theory of practice)
   - 通过实际应用验证内容效果
   - 从创作实践中总结经验规律
   - 持续迭代优化创作流程

4. **群众路线** (Mass line)
   - 以目标受众为中心设计内容
   - 重视用户反馈和接受度
   - 确保内容通俗易懂，贴近用户需求

5. **游击战术** (Guerilla tactics)
   - 快速响应不同的内容需求
   - 灵活调整创作策略适应平台特点
   - 小步快跑，快速迭代内容版本

## 系统架构

### Skill-based 架构优势

系统采用Skill-based架构，相比传统Workflow具有以下优势：

- **模块化设计** - 每个Skill专注单一职责
- **可组合性** - Skill可灵活组合实现复杂流程
- **可进化性** - 随使用积累经验，自我优化
- **可移植性** - Skill为文本文件，易于管理和分享
- **灵活性** - 根据输入动态调整处理流程

### 核心Skills

#### 1. 并行内容创作系列
- `/parallel-copywriting-agents` - 并行文案和插图生成
- `/writing-copywriting-agent` - 专业文案写作
- `/visual-illustration-agent` - 专业视觉设计
- `/comprehensive-copywriting-agent` - 综合文案视觉创作

#### 2. 智能配图系列
- `/intelligent-article-illustrator` - 智能文章配图
- 集成9种预设风格：tech, warm, minimal, playful, notion, elegant, vibrant, retro, nature

#### 3. 小红书专项系列
- `/xiaohongshu-content-agent` - 小红书内容策划
- `/upload-to-xiaohongshu` - 小红书内容发布

#### 4. 代码审核系列
- `/prototype-code-review` - 原型代码轻量审核
- `/production-code-review` - 生产代码严格审核

#### 5. 漫画脚本系列
- `/comic-script-agent` - 文章故事转漫画脚本

## 内容创作流程

### 通用内容创作
```
1. 需求分析 → 2. 文案创作 → 3. 视觉设计 → 4. 智能配图 → 5. 内容整合 → 6. 发布准备
```

### 小红书专项流程
```
1. 输入素材分析 → 2. 内容提纲规划 → 3. 分段文案创作 → 4. 系列信息图设计 → 
5. 视觉元素生成 → 6. 内容整合润色 → 7. 平台适配 → 8. 内容发布
```

### 漫画脚本创作流程
```
1. 原文分析 → 2. 视觉风格确定 → 3. 分镜设计 → 4. 对话框安排 → 5. 脚本生成 → 6. 图像制作
```
