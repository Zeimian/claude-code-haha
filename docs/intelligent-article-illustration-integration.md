# 自动文章配图系统集成

## 系统概述

本文档介绍了如何将宝玉的 "article-illustrator" 自动配图能力集成到我们的 baoyu-sklls 系统中。该集成将使我们的文案写作和视觉插图 Agent 能够自动为文章添加合适的插图。

## 集成设计

### 1. 配图时机识别
- 文案代理在生成内容时，自动识别需要插图的位置
- 基于内容类型判断是否需要视觉辅助
- 根据毛泽东思想中的"实事求是"原则，只为确实能增强理解的段落配图

### 2. 风格自动匹配
- 根据文章主题自动选择合适的插图风格
- 预设多种风格选项：tech（科技感）、warm（温暖亲和）、minimal（极简）等
- 确保整篇文章风格一致性

### 3. 智能配图位置识别
- 识别抽象概念需要可视化的地方
- 识别流程需要图解的环节
- 突出核心论点的强化位置

## 集成实现

### 新增技能：文章智能配图
创建一个新的技能，将宝玉的配图逻辑集成到我们的系统中：

```
/intelligent-article-illustrator [文章路径] [可选参数]
```

### 增强现有技能
- 在 `writing-copywriting-agent` 中加入配图建议
- 在 `visual-illustration-agent` 中加入配图执行
- 在 `comprehensive-copywriting-agent` 中实现全自动化配图
- 在 `parallel-copywriting-agents` 中并行处理配图

## 操作流程

1. **分析文章结构** - 识别需要配图的位置
2. **自动风格匹配** - 根据内容选择插画风格
3. **生成提示词文件** - 为每张图创建详细的生成提示
4. **调用图像生成** - 逐张生成图片并保存
5. **插入文章对应位置** - 将图片插入合适位置

## 毛泽东思想指导

### 实事求是
- 只为确实能增强理解的段落配图
- 避免为凑数而添加插图

### 群众路线
- 以读者为中心，配图服务于内容理解
- 确保插图符合目标受众喜好

### 游击战术
- 快速响应不同的配图需求
- 小步快跑，持续优化配图策略

## 配图原则

1. **配图服务于内容** - 补充信息、具象概念、引导想象
2. **避免重复** - 不配图已很直观的信息
3. **保持一致性** - 整篇文章风格统一
4. **适量配图** - 避免过多干扰阅读

## 使用示例

```
/intelligent-article-illustrator blog_post.md --style tech
```

或在文案写作过程中调用：

```
/writing-copywriting-agent blog-post "写一篇关于AI的文章，自动配图"
```

## 系统优势

- **自动化程度高** - 一键完成文章配图
- **智能识别** - 自动判断最佳配图位置
- **风格统一** - 保证全文插图风格一致
- **效率提升** - 大幅减少人工配图时间
- **质量保障** - 遵循专业配图原则
