# 漫画脚本生成系统

## 系统概述

本系统基于毛泽东思想指导原则，专门用于将文章和故事转换为详细的漫画脚本。系统采用《Logicomix》式的叙事风格，适合呈现科学探索历程、跨越数十年的时间线，以及复杂概念的可视化解释。

## 设计理念

### 毛泽东思想指导原则

1. **群众路线**
   - 将复杂概念转化为群众喜闻乐见的视觉表达
   - 以读者理解为中心设计画面和情节
   - 确保内容既有教育价值又有趣味性

2. **实事求是**
   - 基于原文内容进行准确转化
   - 不歪曲原意，保持内容的真实性
   - 用事实说话，用视觉讲好故事

3. **矛盾论**
   - 识别故事中的主要冲突和转折点
   - 重点突出核心情节和关键概念
   - 合理安排详略，突出重点

## 视觉风格

### 线条清晰（Ligne Claire）
- 参考Hergé《丁丁历险记》或Joost Swarte的画风
- 轮廓线均匀、干净利落
- 背景写实细致，人物略带卡通化但五官特征鲜明

### 色彩分层叙事
- 运用明快的色块区分不同时空场景
- 早期年代采用泛黄的复古暖色调
- 现代科技场景使用冷峻的蓝灰色
- 关键突破时刻以高饱和的亮色强调

### 打破第四面墙
- 设置一位或多位"旁白者"角色
- 他们可以穿梭于画面之间，直接面向读者进行解说
- 形成对话式叙事

## 漫画结构

### 封面设计
- 设计兼具学术厚重感与视觉吸引力的封面
- 标题字体体现知识/科学主题
- 封面构图暗示核心主题
- 明确标注副标题或时间跨度建立史诗感

### 页面布局
- 逐页详细描述封面及每一页的画面布局
- 每页3-6个分镜，合理安排构图
- 区分"主叙事层"与"旁白层"的视觉处理方式
- 为复杂概念预留"概念图解分镜"
- 页面比例为 --ar 2:3

### 叙事结构
- 采用非线性叙事，不同时间线之间跳跃
- 每页末尾设置"钩子"推动翻页欲望
- 关键情节节点安排"定格时刻"

## 技能使用

### 漫画脚本生成代理
```
/comic-script-agent [故事内容] [风格要求]
```

示例：
```
/comic-script-agent "科技创新历程故事" "科学探索主题，现代风格"
/comic-script-agent "人物传记" "复古风格，突出人生转折点"
```

## 输出格式

### 封面页
- **封面设计**: 详细描述封面构图、人物、背景、色彩等

### 内页结构
- **页面X**: 每页详细分镜描述
  - 分镜1: 描述分镜大小、构图、人物动作、场景、对话内容
  - 分镜2: 同上
  - ...
- **页面摘要**: 本页主要内容概括
- **叙事功能**: 推进情节/揭示冲突/解释概念等

### 对话设计
- 主叙事层使用标准椭圆对话框
- 旁白者使用方形或手写风格文字框
- 科学术语首次出现时用特殊标注
- 所有对白使用中文

## 应用场景

1. **知识传记漫画**
   - 科学家传记
   - 技术发展历程
   - 历史事件再现

2. **教育内容**
   - 复杂概念可视化
   - 科学原理阐释
   - 学科历史讲解

3. **科普作品**
   - 科学发现故事
   - 技术创新历程
   - 学术思想演变

## 技术优势

- **忠实原文**: 确保内容准确性
- **视觉优化**: 将抽象概念具象化
- **结构清晰**: 分镜布局合理有序
- **风格统一**: 保持视觉连贯性
- **易于生成**: 提供详细的图像生成指令

通过这套系统，可以高效地将复杂的文字内容转换为生动有趣的漫画脚本，实现知识传播的可视化和大众化。
