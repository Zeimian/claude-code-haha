# 毛泽东思想指导下的小红书内容创作系统

## 系统概述

本系统基于毛泽东思想指导原则，专门针对小红书平台设计内容创作流程。系统通过skill架构将复杂的创作流程分解为可管理的模块，并能够随着使用不断进化。

## 设计理念

### 毛泽东思想指导原则

1. **实事求是** - 根据小红书用户的实际需求创作内容
2. **群众路线** - 以小红书用户为中心设计内容和视觉
3. **矛盾论** - 识别内容创作中的主要矛盾（信息量 vs 可读性）
4. **实践论** - 通过实际发布效果不断优化内容策略
5. **游击战术** - 快速响应不同的内容趋势和用户需求

## 小红书内容创作流程 (Skill架构)

### 第一步：内容分析 (article-analyzer skill)
- 分析用户提供的素材和思路
- 识别内容核心信息点
- 输出 analysis.md

### 第二步：提纲规划 (outliner skill)
- 基于分析结果生成多个提纲方案
- 考虑小红书用户的阅读习惯
- 生成 2-3 个不同风格的提纲方案

### 第三步：内容创作 (writer-agent skill)
- 根据提纲创作小红书风格内容
- 适配小红书的语言风格和表达习惯
- 可并行启动多个创作代理

### 第四步：视觉策划 (xiaohongshu-content-agent skill)
- 拆解内容为1-10张信息图
- 确定视觉风格和排版布局
- 生成图像生成提示词

### 第五步：插图生成 (visual-illustration-agent skill)
- 根据提示词生成手绘风格插图
- 符合小红书审美偏好

### 第六步：内容润色 (polish skill)
- 优化文案和视觉的协调性
- 检查内容是否符合小红书规范

## 技能模块详解

### 1. 小红书内容创作主技能 (`/xiaohongshu-content-agent`)
核心功能：
- 拆解复杂内容为系列信息图
- 设计符合小红书的视觉风格
- 生成图像提示词

### 2. 文案写作技能 (`/writing-copywriting-agent`)
- 为小红书创作人性化、亲和的文案
- 适配小红书的语言风格

### 3. 视觉设计技能 (`/visual-illustration-agent`)
- 为小红书内容生成手绘风格插图
- 保持视觉风格一致性

### 4. 智能配图技能 (`/intelligent-article-illustrator`)
- 为小红书内容自动规划插图位置
- 匹配适合的视觉风格

## 小红书内容创作流程实现

### 文件流转结构
```
input-materials.md → analysis.md → outline.md → draft-content.md → visual-plan.md → final-post.md
```

### 并行处理优化
- 内容创作与视觉策划可并行进行
- 多个视觉方案可并行生成
- 文案与视觉元素同步优化

## 智能上传功能设计

### 小红书平台集成
系统支持将完成的内容直接上传到小红书平台：

```
/upload-to-xiaohongshu [post_file] [credentials]
```

### 上传流程
1. **认证验证** - 验证小红书账户凭据
2. **内容格式化** - 按小红书要求格式化内容
3. **图片上传** - 上传生成的插图到小红书
4. **发布操作** - 发布完整内容到小红书

### 安全考虑
- 使用加密方式存储账户信息
- 提供安全的认证流程
- 遵循小红书的API使用规范

## 可进化特性

### 自我优化机制
- 记录内容表现数据（点赞、收藏、评论）
- 分析高表现内容的共同特征
- 自动调整创作策略和视觉风格

### 学习机制
- 从每次发布后的用户反馈中学习
- 持续优化内容创作的提示词
- 根据平台趋势调整视觉风格

## 实际应用示例

### 完整工作流
```
1. /xiaohongshu-content-agent beauty-tips "创建美容护肤技巧内容"
2. /visual-illustration-agent beauty "为美容内容生成插图"
3. /writing-copywriting-agent beauty-advice "撰写护肤建议文案"
4. /intelligent-article-illustrator beauty-content.md "为内容智能配图"
5. /upload-to-xiaohongshu final-beauty-post.md "上传到小红书"
```

### 并行处理示例
```
/parallel-copywriting-agents xiaohongshu-beauty "同时创作文案和规划视觉"
```
