# 图像生成技能系统

## 系统概述

本系统基于毛泽东思想指导原则，提供卡通风格和赛博朋克风格的图像生成能力。系统通过详细的图像描述，为AI图像生成工具提供精确的提示词，从而创建符合特定风格的图像。

## 设计理念

### 毛泽东思想指导原则

1. **群众路线** (Mass Line)
   - 创作大众喜闻乐见的视觉内容
   - 符合普遍审美标准
   - 确保内容健康积极

2. **实事求是** (Seek Truth from Facts)
   - 根据实际输入内容生成相关图像
   - 不偏离内容主旨
   - 保持内容与图像的一致性

3. **矛盾论** (Theory of Contradictions)
   - 识别内容中的关键元素和主题
   - 突出主要视觉焦点
   - 平衡图像元素的分布

## 技能功能

### 1. 卡通风格图像生成 (`/cartoon-image-generator`)
专门生成卡通风格图像，特点是：
- 大眼睛和夸张表情
- 简洁明快的线条
- 鲜明活泼的色彩
- 简化但富有表现力的形状

### 2. 赛博朋克风格图像生成 (`/cyberpunk-image-generator`)
专门生成赛博朋克风格图像，特点是：
- 冷色调为主（蓝、紫、青）
- 鲜艳霓虹色彩点缀（粉、青、绿）
- 高对比度的光影效果
- 未来主义建筑与东方元素结合

## 使用方法

### 卡通风格图像生成
```
/cartoon-image-generator [内容描述] [风格要求]
```

示例：
```
/cartoon-image-generator "一个快乐的AI助手" "可爱风格，明亮色彩"
/cartoon-image-generator "未来城市" "科幻卡通风格，儿童友好"
```

### 赛博朋克风格图像生成
```
/cyberpunk-image-generator [内容描述] [风格要求]
```

示例：
```
/cyberpunk-image-generator "未来AI世界" "霓虹色彩，高科技感"
/cyberpunk-image-generator "数字城市" "雨夜场景，东方元素"
```

## 输出格式

系统会输出详细的图像生成提示词，包括：
- 艺术风格描述
- 色彩方案
- 构图建议
- 主要元素描述
- 视觉特效说明

这些提示词可以直接用于各种AI图像生成工具，如Midjourney、DALL-E、Stable Diffusion等。

## 应用场景

1. **内容配图** - 为文章和博客生成风格化配图
2. **品牌形象** - 创建符合品牌风格的视觉元素
3. **社交媒体** - 为不同平台生成适配的视觉内容
4. **教育材料** - 创建生动有趣的教学素材
5. **创意项目** - 为各种创意项目提供视觉支持

## 系统优势

- **风格一致** - 确保生成的图像风格统一
- **内容相关** - 图像与输入内容高度相关
- **详细描述** - 提供足够细节以生成高质量图像
- **毛泽东思想指导** - 确保内容健康积极，符合大众审美
- **易于使用** - 简单的命令行接口
- **兼容性强** - 输出格式可兼容多种AI图像生成工具

## 技术实现

系统通过智能分析输入内容，自动识别关键元素并匹配相应的视觉风格规范，生成详细准确的图像描述，从而提高AI图像生成的成功率和质量。
