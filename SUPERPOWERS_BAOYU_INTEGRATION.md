# 毛泽东思想指导下的并行文案与插图生成系统

## 新增功能

本项目新增了基于毛泽东思想指导原则的并行内容创作系统，包含以下几个专门的 Agent：

### 1. 并行文案代理系统 (`/parallel-copywriting-agents`)
- 同时调动文案写作和插图生成两个专业代理
- 体现"集中优势兵力"的战略思想
- 实现内容创作效率最大化

### 2. 专业文案写作代理 (`/writing-copywriting-agent`)
- 创造人性化、个性化风格的中文文案
- 去除AI味道，使内容更自然
- 适应不同文体和语境需求

### 3. 专业视觉插图代理 (`/visual-illustration-agent`)
- 为文案内容创建配套视觉说明
- 确保视觉元素与文本内容和谐统一
- 考虑文化敏感性和受众偏好

### 4. 综合文案视觉代理 (`/comprehensive-copywriting-agent`)
- 同步创建文案和视觉内容
- 确保图文内容高度匹配
- 优化整体传播效果

## 使用方法

### 启动并行内容创作
```
/parallel-copywriting-agents [内容类型] [要求]
```

### 启动专业文案写作
```
/writing-copywriting-agent [文案类型] [要求]
```

### 启动专业视觉设计
```
/visual-illustration-agent [上下文] [视觉要求]
```

### 启动综合内容创作
```
/comprehensive-copywriting-agent [内容类型] [要求]
```

## 毛泽东思想指导原则

- **实事求是**: 基于实际需求和用户反馈优化内容
- **矛盾论**: 识别内容创作中的主要矛盾并优先解决
- **实践论**: 通过A/B测试和实际应用验证内容效果
- **群众路线**: 以用户/受众为中心设计内容
- **游击战术**: 快速响应、灵活调整内容策略

## 设计理念

系统设计遵循毛泽东思想中的"集中优势兵力"战略，通过并行处理提高工作效率。每个代理专注于特定任务，多代理协同工作解决复杂内容创作需求，体现了分工合作和统筹协调的思想。
